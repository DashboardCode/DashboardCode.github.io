BsMultiselect

Multiselect Plugin for Bootstrap 5

dashboardcode.BsMultiSelect("#myElement");
$("#myElement").bsMultiSelect(); // jquery


How to:

Add columns to css or css-from-js:

                dashboardcode.BsMultiSelect("#myElement",
                    {  
                        cssPatch : { 
                            choices: {
                                columnCount:'4' 
                            }
                        }
                    }
                );
