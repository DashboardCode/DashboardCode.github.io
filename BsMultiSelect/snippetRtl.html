Multiselect JQuery Plugin for Bootstrap 4

Step (1): add dir="rtl" into DOM chierarchy, e.g.:

<form dir="rtl">...</div>

Step (2): add rtl support to used bootsrap components (custom checkboxes, form-control validation); most probably if you use BS with RTL this step is allready done and your css contains something like this:

<style>
[dir="rtl"] .custom-control.custom-checkbox{
    margin-left: 0;
    margin-right: 1.5rem;
}
[dir="rtl"] .custom-control.custom-checkbox .custom-control-label::after,
[dir="rtl"] .custom-control.custom-checkbox .custom-control-label::before{
    left: auto;
    right: -1.5rem;
}
.was-validated .form-control:valid, .form-control.is-valid{
    padding-right: 0.75rem;
    padding-left: calc(1.5em + 0.75rem);
    background-position: left calc(0.375em + 0.1875rem) center;
}    
</style>
