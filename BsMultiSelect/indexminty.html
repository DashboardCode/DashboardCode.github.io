DashboardCode Multiselect Plugin for Bootstrap 4

Just Bootstrap 4 plugin

$("select[multiple='multiple']").bsMultiSelect();

Since default disabled background color and focus colors were changed by Minty theme:

                <script>
                     $("select[multiple='multiple']").bsMultiSelect({
                        selectedPanelDisabledBackgroundColor:"#f7f7f9",
                        selectedPanelFocusBorderColor: '#d0eae2',
                        selectedPanelFocusBoxShadow: '0 0 0 0.2rem rgba(120,194,173,0.25)',
                        selectedPanelFocusValidBoxShadow: '0 0 0 0.2rem rgba(86,204,157,0.25)',
                        selectedPanelFocusInvalidBoxShadow: '0 0 0 0.2rem rgba(255,120,81,0.25)'
                     });
                </script>

Additionally styles should be patched with this css since original Minty theme change default "not colored" badge text color to white and it become invisible placed on form-control:

                <style> 
                    .form-control .badge {color:black}
                </style>
