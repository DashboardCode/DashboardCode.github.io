Since default focus border+shadow and disabled background color were changed by Sketchy theme:

                    <link rel="stylesheet" href="https://rawgit.com/DashboardCode/BsMultiSelect/master/dist/css/BsMultiSelect.css">
                    <style>
                            /* alternatively you can make a copy BsMultiSelect.css and change values in place */
                            .dashboardcode-bsmultiselect > ul.form-control.focus:not(.is-valid):not(.is-invalid){
                               border-color:#333;
                               box-shadow: 0px 0px 0px 0.2rem rgba(51,51,51,0.25)
                            }
                            .dashboardcode-bsmultiselect > ul.form-control.disabled{
                                background-color: rgb(247, 247, 249);
                            }
                    <style>

Also styles should be patched with css since original Sketchy theme has no highlighting on hover in dropdown-menu:

                    <style>
                    .dashboardcode-bsmultiselect > ul.dropdown-menu > li.bg-light {
                        background-color:rgb(240, 240, 240) !important;
                    }
        
                    .dashboardcode-bsmultiselect > ul.dropdown-menu > li.text-primary  label{
                        color:rgb(99, 97, 97);
                    }
                    </style>
